# university_works
В этом репозитории представлены проекты по программированию, алгоритмам и структурам данных, построению и анализу алгоритмов, ООП.

**#Обработка изображения (C, libpng, getopt, stdlib)**

Программа представляет собой CLI утилиту, обрабатывающую png-изображение с помощью библиотеки libpng.

Доступны следующие функции:
1. Рисование квадрата с заданными параметрами;
2. Перестановка четырех частей выбранной прямоугольной области в заданном пользователем порядке;
3. Замена наиболее часто встречающегося цвета на заданный.
4. Создание контрастного изображения;
5. Заливка изображения заданным цветом за исключением выделенной прямоугольной области;
6. Циклический сдвиг изображения;
7. Рисование ромба заданного цвета;
8. Выделение рамкой пикселей с заданным цветом;
9. Отражение по диагонали.

Мануал по флагам и технический отчет с подробным описанием утилиты представлен в директории *cw2_adds*.

**#Игра "Cat & Mice" (C++, SFML, nlohmann/json, CMake)**

Пошаговая стратегическая 2D игра с элементами RPG, механиками карточных заклинаний и системой сохранения.

Ключевые особенности:
* ООП-архитектура: разделение логики игрового движка, GUI-компонентов и сущностей (игрок, враг, поле, заклинания).
* Игровой процесс: перемещение по полю с препятствиями и замедляющими клетками и уничтожение врагов (подробнее [здесь](https://github.com/alina-kiseleva/university_works/releases/tag/v1.0.0)).
* Боевая система: ближний и дальний бой (*"E"* и *"Q"*), карточки заклинаний.
* Прогресс и сериализация: улучшения при переходе на следующий уровень, сохранение прогресса в JSON-файл и дальнейшая загрузка для продолжения начатой игры.
* Безопасность: хэш-функция для проверки корректности данных в JSON-файле.

Детальное рассмотрение программной реализации представлено в отчетах в директории *game_cat_and_mice_reports*.

Запуск: подробнее в [**Releases**](https://github.com/alina-kiseleva/university_works/releases/tag/v1.0.0).

**#Система управления онлайн-заказами в кафе (C++, CMake)**

Программа реализует два режима работы: администраторский и клиентский, предоставляя функционал для управления меню блюд и формирования заказов.

*Режим администратора:*
* Добавление, изменение и удаление позиций меню и цен в реальном времени.
* Требует авторизации (пароль *1503*).

*Режим клиента:*
* Просмотр меню, добавление и удаление позиций в корзине, рассчет итоговой стоимости заказа.
* Запускается по умолчанию.

Запуск: ./cw3 

**#Алгоритмы (algorithms)**

Алгоритмы и структуры данных (АиСД) -- реализация сортировки и структур данных:
* TimSort
* Circular deque (кольцевая дека)
* Rope
* Segment tree (дерево отрезков)

Построение и анализ алгоритмов (ПиАА) -- реализация алгоритмов:
* Backtracking (задача о квадрировании квадрата)
* TSP (задача коммивояжера, точное и приближенное решения)
* Knuth–Morris–Pratt algorithm (поиск подстроки в строке)
* Aho-Corasick algorithm (поиск набора подстрок в тексте)
