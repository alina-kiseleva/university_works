**Инструкция по использованию консольной утилиты обработки изображения**

Компиляция: gcc cw_2sem.c -o cw -lpng
Синтаксис: ./cw [FUNCTION]... [FILE]

Общая информация:
* Исполняемый файл и файл входного изображения должны находиться в одной директории.
* Последним аргументом утилите должно передаваться имя входного png файла.
* По умолчанию имя выходного файла out.png.
* За один запуск выполняется только одна функция.
* При вызове функции порядок введенных флагов не важен.

ОСНОВНЫЕ ФУНКЦИИ:
    --square, -S
        функция рисования квадрата. Вызывается со следующими флагами:

        --left_up, -l
            координаты левого верхнего угла квадрата. Значение задается в формате [x.y] (без скобок).

        --side_size, -s
            размер стороны, положительное число.

        --thickness, -t
            толщина линий, положительное число.

        --color, -C
            цвет линий. Значение задается в формате [rrr.ggg.bbb] (без скобок), где на каждой позиции находится число в диапазоне [0; 255], задающее соответствующую цветовую компоненту.

        --fill, -f
            наличие заливки. Если флаг введен, то квадрат будет залит.

                --fill_color, -c
                     цвет заливки. Работает так же, как и --color.

        ПРИМЕР ВЫЗОВА: ./cw --square --left_up 52.67 --side_size 77 --thickness 3 --color 100.13.42 --fill --fill_color 255.255.255 test.png

    --exchange, -e
        функция перемещения частей прямоугольной области изображения. Вызывается со следующими опциями:

        --left_up, -l
            координаты левого верхнего угла области. Значение задается в формате [x.y] (без скобок).

        --right_down, -r
            координаты правого нижнего угла области. Работает аналогично флагу --left_up.

        --exchange_type, -T
            способ обмена частей. Возможные значения:
                clockwise           обмен по часовой стрелке,
                counterclockwise    обмен против часовой стрелки,
                diagonals           обмен по диагоналям.

        ПРИМЕР ВЫЗОВА: ./cw --exchange --left_up 52.10 --right_down 123.158 --exchange_type clockwise test.png

    --freq_color, -F
        функция замены наиболее часто встречаемого цвета да заданный цвет. Вызывается со следующими флагами:

        --сolor, -C
            цвет, на который будет заменен самый частый. Значение задается в формате [rrr.ggg.bbb] (без скобок), где на каждой позиции находится число в диапазоне [0; 255], задающее соответствующую цветовую компоненту.

        ПРИМЕР ВЫЗОВА: ./cw --freq_color --color 255.255.255 test.png

ДОПОЛНИТЕЛЬНЫЕ ФУНКЦИИ:
    --contrast, -R
        функция, меняющая значение каждой компоненты пикселя по формуле new = alpha * beta. Вызывается со следующими флагами:

        --alpha, -a
            коэффициент изменения контрастности, положительное число с плавающей точкой.

        --beta, -b
            сдвиг контрастности, целое число.

        ПРИМЕР ВЫЗОВА: ./cw --contrast --alpha 9.5 --beta 3 test.png

    --shift, -H
        функция, реализующая циклический сдвиг изображения. Вызывается со следующими флагами:

        --step, -p
            значение циклического сдвига.

        --axis, -x
            ось, по которой будет произведен сдвиг. Принимает значения 'x', 'y', 'xy'.

        ПРИМЕР ВЫЗОВА: ./cw --shift --step 100 --axis xy test.png

    --outside_rect, -O
        функция, которая закрашивает изображение вне указанной области в определенный цвет. Вызывается со следующими флагами:

        --left_up, -l
            координаты левого верхнего угла области. Значение задается в формате [x.y].

        --right_down, -r
            координаты правого нижнего угла области. Работает аналогично флагу --left_up.

        --color, -C
            цвет в формате [r.g.b], в который необходимо закрасить изображение вне указанной области.

        ПРИМЕР ВЫЗОВА:  ./cw --outside_rect --left_up 78.99 --right_down 358.201 --color 128.0.100 test.png

    --square_rhombus, -U
        функция, рисующая закрашенный равносторонний ромб. Вызывается со следующими флагами:

        --upper_vertex, -v
            координаты верхней вершины ромба. Значение задается в формате [x.y].

        --size, -z
            размер стороны ромба, положительное число.

        --fill_color, -c
            цвет заливки ромба в формате [r.g.b].

        ПРИМЕР ВЫЗОВА: ./cw --square_rhombus --upper_vertex 128.10 --size 79 --fill_color 14.61.30 test.png

    --diag_mirror, -M
        функция, которая отражает заданную область по диагонали. Вызывется со следующими флагами:

        --left_up, -l
            координаты левого верхнего угла области. Значение задается в формате [x.y].

        --right_down, -r
            координаты правого нижнего угла области. Работает аналогично флагу --left_up.

        ПРИМЕР ВЫЗОВА: ./cw --diag_mirror --left_up 50.50 --right_down 200.200 test.png

    --circle_pixel, -I
        функция, которая обводит все пиксели заданного цвета в рамку. Вызывается со следующими флагами:

        --color, -C
            цвет в формате [r.g.b], пиксели которого будут обведены в рамку.

        --size, -z
            размер стороны рамки, положительное число.

        --circle_color, -e
            цвет рамки в формате [r.g.b].

        ПРИМЕР ВЫЗОВА: ./cw --circle_pixel --size 5 --color 0.0.0 --circle_color 0.150.0 test.png
        
ОБЩИЕ ФУНКЦИИ:
    --input, -i
        функция задания имени входного изображения. При ее использовании, в конце скрипта имя входного файла не указывается.

    --output, -o
        функция задания имени выходного изображения.

    --info [FILE] 
        функция, показывающая информацию о входном изображении.

    --help, -h
        функция вывода справки, содержащей информацию об основных функциях и их флагах.
